Downloading and installing Umple

Below are instructions for downloading and installing Umple, a model-oriented programming technology that lets you model in code. Here is more information about Umple, or try it online without downloading, using the UmpleOnline tool

A. Requirements to run Umple

Umple can be used on Mac-OS, Windows or Linux.

For Eclipse

  • Eclipse with Modelling tools. It is known to work under Indigo, Juno or Kepler.

For command-line use

  • An up-to-date JVM. Java 7 recommended.
 

B. Directions

B.1 Downloading Umple (choose from the following options)

  • The latest official releases of Umple jars at GitHub
     
    The command-line compiler and the Eclipse plugin are available, here along with release notes.
     
  • The latest build of the command-line compiler at the University of Ottawa
     
    This is built after every commit from our continuous build process. This is also the version of the compiler used by UmpleOnline. This version may contain experimental features so should not be used in critical applications.
     
  • To obtain the full tree with source code (including the Umple compiler as a jar, which is necessary to compile Umple itself, select one of the following:
     
    • Definitive Subversion Master With Lastest Software: svn checkout http://umple.googlecode.com/svn/trunk/ umple-read-only
    • UOttawa git mirror (updated automatically after every commit once the continuous integration server has verified that all tests pass): git clone http://cruise.site.uottawa.ca/git/umple-svn-git-mirror/.git
    • Github mirror main page: git clone https://github.com/umple/Umple.git
    • SourceForge mirror main page
    • Release tarballs or zip files, available at Github as described above.

     
  • Recent changes: The list of completed changes applied in the last six months can be found by clicking here

B.3 Running the Umple compiler on the command line

  • Run 
    java -jar umple.jar <options> <arguments>
  • Specify your umple file as an argument. 
  • You can use --help as an option to obtain the full set of command-line options. Options available include:
    • -g, --generate : Specify the output language: Java,RTCpp, Php,RTCpp,Ruby,SQL,Json,Ecore,TextUml, Yuml,Cpp etc.
    • --help : Display the help message
    • --override : If a output language is specified using option -g, output will only generate language
    • --path : If a output language is specified using option -g, output source code will be placed to path
    • -v, --version : Print out the current Umple version number
