MultiValued Attributes
Attributes
noreferences

@@description
<p>As in UML, you can have attribute with multiple values.</p>

<p>We encourage the use of association notation in this context, however
the attribute notation can be useful sometimes</p>

@@syntax

[[complexAttribute]]

@@example
class Office {
   Integer number;
   Phone[] installedTelephones;
} 

class Phone {
  String digits;
  String callerID;
}

@@endexample
