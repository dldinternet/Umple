W050 Target State Not Found
Errors and Warnings
noreferences

@@description

<h2>Umple semantic warning reported when the target state of a transition is not listed.</h2>

<p>
An end state (with no outgoing transitions) is generated. The purpose of the warning is to alert the developer that they may have a typographical error in the target state name.</p>


@@example
// This example generates the error message
class X {
  sm {
    s1 {
      e1 -> s2;
    }
  }
}

@@endexample

@@example
// This example solves the problem
class X {
  sm {
    s1 {
      e1 -> s2;
    }
    s2 {}
  }
}

@@endexample
