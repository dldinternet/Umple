E216 Twice Binding of a Parameter
Errors and Warnings
noreferences

@@description

<h2>Umple semantic error related to binding a template parameter twice</h2>

<p>When using traits, we can bind values for template parameters. In the process of binding, we can bind two values for a template parameters. This bring a case in which there is no clear binding. The Umple compiler detects this case and prevents the system from being complied. </p>
