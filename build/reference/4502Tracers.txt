Tracing

@@description
<p>Data collected while tracing ranges from a simple logging mechanism into a more sophisticated tracing mechanism that involves the record of lower level events triggered in user and kernel spaces. Umple provides tracing using different tracers. Tracers can be categorized into two main categories: </p>

<p>
<ol>
<li>Primitive:</li>
  <ul>
  <li>Console</li>
  <li>File</li>
  </ul>
&nbsp;<br />
<li>Advanced (under development):</li>
   <ul>
   <li>LTTNG </li>
   <li>Dtrace </li>
   </ul>
</ol>
</p>

@@syntax

[[traceType]]

@@example

// this example generates traces using the File tracer
tracer File;

class Student
{
  Integer id;
  trace id;
}

@@endexample
