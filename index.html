Umple - The Model-Oriented Programming Technology

Umple is a technology to allow developers to add modeling abstractions to their code, or to start with a textual model and add code embedded in that model. Umple supports UML class and state diagrams as well as Java, PhP and C++. You can try it online at UmpleOnline.

The master repository for Umple is currently hosted at Google Code and mirrored to GitHub.
